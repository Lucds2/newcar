# 🚗 Plataforma de Venda de Veículos | Web

Uma aplicação web moderna, limpa e responsiva desenvolvida para funcionar como uma vitrine virtual de uma concessionária de carros. O site foi projetado para expor o catálogo de veículos disponíveis, destacando especificações técnicas essenciais e facilitando o contato direto de potenciais compradores.

---

## 🚀 Funcionalidades

* **Vitrine de Veículos:** Exibição estruturada de automóveis organizados em cards individuais com imagens de alta qualidade.
* **Ficha Técnica Rápida:** Destaque visual das principais características de cada carro, como:
  * Ano de fabricação
  * Quilometragem atual (KM)
  * Tipo de transmissão (Manual ou Automático)
  * Preço de venda em destaque
* **Integração de Contato:** Botões de ação direta voltados para conversão, direcionando o cliente interessado para os canais de atendimento e WhatsApp.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi estruturado seguindo padrões modernos e performáticos de desenvolvimento Front-End nativo:

* **HTML5:** Estruturação semântica de longa escala (`index.html`), utilizando tags apropriadas para seções, listas de especificações e blocos repetíveis de produtos (cards).
* **CSS3:** Gerenciamento de layouts fluidos, estilização e alinhamento dos componentes da vitrine para garantir uma navegação agradável.

---

## 📂 Estrutura de Arquivos

* `/imagens`: Diretório contendo todas as fotos dos veículos cadastrados no catálogo e ícones de interface.
* `index.html`: Arquivo principal contendo a estrutura da página, o cabeçalho descritivo, a grade de carros e o rodapé institucional.
* `style.css`: Folha de estilos responsável pelo design visual, tipografia, cores e responsividade da loja virtual.
